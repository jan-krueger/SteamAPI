## SteamAPI Wrapper & More

Documentation provided by [xpaw](https://lab.xpaw.me/steam_api_documentation.html).

## Usage
```xml
[...]
<dependency>
    <groupId>de.sweetcode</groupId>
    <artifactId>SteamAPI</artifactId>
    <version>1.0.0-alpha</version>
</dependency>
[...]
```
